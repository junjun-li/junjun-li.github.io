# 了解 HTTP 协议

# 为什么在浏览器输入 URL 之后，就能访问 Web 页面？

域名：如 www.baidu.com 等就像一个名字，在互联网上，是使用IP来进行标识的。我们访问的其实是IP。

(opens new window)

# 浏览器与服务器的交互流程

  1. 浏览器通过地址栏发出请求
  2. 通过 DNS 服务器解析，得到域名对应的 ip 地址
  3. 根据 ip 地址，访问服务器具体的某个文件
  4. 服务器响应这个具体的文件
  5. 浏览器获取响应，进行显示
